Now, we can compute the posterior quanitites for each gene snp pair. Here, well test it on 100 gene snp pairs.

weightedquants=lapply(seq(1:100),function(j){total.quant.per.snp(j,covmat,b.gp.hat=b.test,se.gp.hat = se.test,pis,A="Simulations",checkpoint = FALSE)})

Note that we create a file of posterior weighted means, lfsr and marginal variances.

To make sure that these computations are correct,
